# 🎮 Task 2 – Tic-Tac-Toe AI (Minimax Algorithm)

This project is part of the **CodSoft Artificial Intelligence Internship**.

## 📌 Objective
Create an unbeatable AI player for the game of Tic-Tac-Toe using the **Minimax algorithm**.

## ⚙️ Tech Stack
- Python
- Minimax Algorithm
- Console Input/Output

## 🧠 Features
- Human plays as 'X', AI as 'O'
- AI plays the optimal move using recursive Minimax
- Detects win/loss/draw scenarios
- Turn-by-turn interaction in terminal

## 🚀 How to Run
1. Clone the repo or download the `.py` file
2. Run the game using Python 3:
```bash
python task2_tic_tac_toe_ai.py
